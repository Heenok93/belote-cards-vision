# Documentation du projet

## Belote Cards Vision est une application développée en Python et Streamlit permettant d'analyser automatiquement une photo de cartes de Belote grâce à un modèle de Deep Learning (YOLO), puis de calculer le score de la manche selon les règles officielles. L'application intègre également des outils de correction manuelle, de prétraitement d'image et un historique des scores via SQLite.
